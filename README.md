## Projects

### IAM User Activity Scanner
[View Project](iam-scanner/)

Python script that scans all IAM users in an AWS account and reports their last activity (console login or API key usage). Built with boto3 to automate security audits and identify stale credentials.

*Key features:*
- Lists all IAM users
- Checks PasswordLastUsed for console activity
- Checks GetAccessKeyLastUsed for API activity
- Outputs a clean report
